AzurImmo - Kotlin

Le projet intervient lors de la seconde année de formation. L'objectif est de découvrir le développement mobile sur Android

Fonctionnalités

  • Lister les batiments qui contiennent des immeubles
  • Supprimer les bâtiments
  • Modifier les bâtiments
  • Ajouter des bâtiments
  • Lister les appartements liés aux bâtiments
  • Supprimer seulement les appartements
  • Modifier seulement les appartements
  • Ajouter des appartements

Technologies utilisées

API REST en Java Kotlin
